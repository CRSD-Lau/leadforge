## Quick Start

### 1. Website

```bash
cd website
npm install
npm run dev          # вҶ’ http://localhost:3000
npm run build        # Production build
```

Deploy to Vercel:
```bash
npx vercel           # Follow prompts
```

### 2. Agent вҖ” Local Development

```bash
cd agent
python -m venv .venv
source .venv/bin/activate    # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env         # Fill in your API keys
python agent.py
```

### 3. GitHub Actions (Production)

Add these as **repository secrets** (Settings вҶ’ Secrets вҶ’ Actions):

| Secret | Where to get it |
|--------|----------------|
| `ANTHROPIC_API_KEY` | console.anthropic.com |
| `RESEND_API_KEY` | resend.com |
| `GOOGLE_MAPS_API_KEY` | Google Cloud Console вҶ’ Places API |
| `SENDER_EMAIL` | neil@leadforge-ai.ca |
| `GMAIL_TOKEN_JSON` | Generated by Gmail OAuth flow |

The workflow fires MonвҖ“Fri at 9:00 UTC (вүҲ6:00 AM ADT).  
Manual trigger available via the **Actions** tab вҶ’ **Run workflow**.

## Gmail OAuth Setup (for reply handling)

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Enable the **Gmail API**
3. Create OAuth 2.0 credentials вҶ’ Desktop App
4. Download `credentials.json` вҶ’ place in `agent/`
5. Run once locally to generate `token.json`:
   ```bash
   python -c "from reply_handler import check_and_handle_replies; check_and_handle_replies()"
   ```
6. Copy the contents of `token.json` вҶ’ GitHub Secret `GMAIL_TOKEN_JSON`
